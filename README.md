## To Run application locally

1. Configure environment variables in a `.env` file in the root directory:
```
    AWS_ACCESS_KEY_ID=...
    AWS_SECRET_ACCESS_KEY=...
    S3_BUCKET_NAME=...
    AWS_REGION=...

    AZURE_OCR_ENDPOINT=...
    AZURE_OCR_SECRET_KEY=...
```

2. Run docker containers:
```bash
    docker-compose up --build
```
